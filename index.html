HTML

HTML, or HyperText Markup Language, is the standard language used to create and design web pages. It's the foundation of web development, allowing developers to structure content on the web.

CSS

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall visual appearance of web pages.

JavaScript

JavaScript is a versatile, high-level programming language commonly used to create interactive effects and dynamic web pages. It is one of the core technologies of the World Wide Web, alongside HTML and CSS.
